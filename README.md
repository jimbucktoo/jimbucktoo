# Project Title

jimbucktoo.com is a website developed as a portoflio site to showcase the projects that Web Developer James Liang has completed.

## Built With

* [Materialize.css](https://materializecss.com) - The material design front-end framework used.
* [Materialize Icons](https://material.io/tools/icons/?style=baseline) - Materialize icons.
* [jQuery](https://developers.google.com/speed/libraries/#jquery) - JavaScript library designed to simplify the client-side scripting of HTML.
* [Google Firebase](https://firebase.google.com/) - Hosting.
* [ScrollReveal](https://scrollrevealjs.org) - Used to create scroll revealing effect.
* [Sketch App](https://www.sketchapp.com) - Used to create images & graphics.
* [Google Fonts](https://fonts.google.com) - Used for fonts.
